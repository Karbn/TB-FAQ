Flags
If you want to change multiple game rules at once, flags are great to use.
A flag is a set combination of DQ, DM, Frac and Grip settings.
You can set a flag in the game rules (ctrl+g) or using the command /set flags xx.

Here is a list of all flags:

E.g. you want to play a game with DQ, DM and Frac, but without grip, you type /set flags 15 ingame.
