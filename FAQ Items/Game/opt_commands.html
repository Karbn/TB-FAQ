Opt Commands

Q: What are opt commands?

A: Opt is shorthand for 'option'. As such, the opt commands are simply a way of changing options through text commands rather than through a graphical interface (like a menu).



Q: What options can be changed this way?

A: There are a lot of options which can be changed through the use of opt commands. Almost anything which is controllable through the client on your computer can be changed via the use of these commands.

Here's the complete list of opt commands available to you (default values are the numbers proceeding the commands):

Options:
                    
    
    COMMAND             RANGE             DEFAULT             DESC
    
   -altimeter           [0|1]             0                   Display the altimeter (current height + record height)
   -speedometer         [0|1]             0                   Display the speedometer (current speed + max speed)
   -hud                 [0|1]             1                   Additional HUD elements (scores, names, chat etc.)
   -gui                 [0|1]             1                   -
   -score               [0|1]             1                   Display scores
   -timer               [0|1]             1                   Display timer
   -chat                [0|1]             1                   Display chat
   -spectator           [0|1]             1                   Display spectators in queue
   -debug               [0|1]             0                   Debug information
   -name                [0|1]             1                   Display names
   -emote               [0|1]             1                   Show emotes (/em)
   -shout               [0|1]             1                   Show shouts (/sh)
   -button              [0|1]             0                   -
   -text                [0|1]             1                   Display full user text (name + belt + rank etc.)
   -hint                [0|1]             1                   Show hints ('Press spacebar', etc.)
   -feedback            [0|1]             1                   Show feedback ('Extending Left Hip' etc.)
   -message             [0|1]             1                   Enable end-of-match message ('Tori won' etc.)
   -info                [0|1]             1                   -
   -cursor              [0|1]             0                   -
   -uke                 [0|1]             1                   Draw uke (will be invisible if 0)
   -tori                [0|1]             1                   Draw Tori (will be invisible if 0)
   -blood               [0|1]             1                   Display Blood
   -sound               [0|1]             1                   -
   -mousebuttons        [0-3]             3                   1 = click can scroll all joint modes | 2 = click can only scroll hold/lax modes | 3 = click can only scroll extend/contract modes
   -backgroundclick     [0|1]             1                   Allows clicking empty space to un-focus from a specific player
   -rumble              [0|1]             0                   -
   -smartcam            [0|1]             0                   -
   -width               [0-unbounded]     960                 Sets the width of the client (in pixels)
   -height              [0-unbounded]     540                 Sets the height of the client (in pixels)
   -antialiasing        [0|1]             1                   Enables anti-aliasing
   -benchmark           [0|1]             0                   Enables benchmarking
   -fullscreen          [0|1]             0                   Enables fullscreen mode
   -autosave            [0|1]             1                   When on, all replays viewed will automatically be saved
   -rememberrules       [0|1]             0                   When on, current game rules will be remembered for next time you load up the client
   -autoupdate          [0|1]             1                   When on, player data will be updated on the fly
   -bruise              [0|1]             1                   Display bruising on joints (redness when taking damage)
   -motionblur          [0|1]             0                   Enables motion blur
   -bloodstains         [0|1]             1                   Draw blood stains on the floor (requires shaders)
   -exhibition          [0|1]             0                   Begins replay slideshow
   -reflection          [0|1]             1                   Draw reflections (requires shaders)
   -smoothcam           [0|1]             1                   Enables smooth camera
   -framerate           [30|60]           30                  Changes framerate cap to either 30 or 60 (other values will be accepted, however only 30 and 60 can be set). You can uncap your framerate using the fixedframerate option.
   -fixedframerate      [0|1]             1                   Allows for as many frames per second as your environment will allow for.   
   -avatar              [0|1]             1                   Display head avatars beneath user info
   -fluid               [0|1]             1                   Activate fluid blood
   -softshadow          [0|1]             1                   Draw soft shadows
   -ambientocclusion    [0|1]             1                   Enable ambient occlusion
   -bumpmapping         [0|1]             1                   Enable bumpmapped textures
   -depthoffield        [0|1]             0                   Enables depth of field
   -anaglyphic          [0|1]             0                   Enables analgyphic mode
   -shaders             [0|1]             0                   Enables shaders (required for some other options to work)
   -noreload            [0|1]             1                   -
   -equipment           [0|1]             1                   -
   -money               [0|1]             1                   Display current toricredits
   -turnalert           [0|1]             1                   When on, will cause the client to issue an alert when it is the players turn
   -beginner            [0|1]             1                   -
   -animatedhead        [0|1]             0                   -
   -floortexture        [0|1]             1                   Draw floor texture items        
   -particles           [0|1]             1                   Draw flame particles
   -trails              [0|1]             1                   Draw trail items
   -keyframes           [0|1]             0                   Enables keyframed camera movement in replays
   -ghostcache          [0|1]             0                   Enables caching of player ghosts
   -playercache         [0|1]             0                   Enables caching of player data
   -chatcache           [0|1]             1                   Enables caching of viewed chat
   -raytracing          [0|1]             1                   Enables raytracing (requires shaders)
   -dl                  [0|1]             1                   Enables on-the-fly downloading of other players data
   -hair                [0|1]             0                   Draw hair items
   -grip                [0|1]             0                   Draw grip spheres

Some of these options may not work in certain versions of Toribash
